# Dev Skill 구현

## 개요
개발자 대시보드 `/dev` skill을 구현했다. 개발자별 작업 현황, 커밋/PR 활동, 스프린트 기여도, 팀 비교 기능을 제공한다.

## 주요 기능

### 1. `/dev` 또는 `/dev me`
- 현재 스프린트 할당 이슈
- 최근 7일 커밋 활동
- 오픈 PR 목록

### 2. `/dev <name>`
- 특정 개발자 대시보드
- Vision 역할 정보
- 스프린트 할당 이슈
- 커밋 활동

### 3. `/dev commits`
- 팀원별 커밋 활동 분석
- `--days N` 옵션으로 기간 지정
- 레포지토리별 활동

### 4. `/dev prs`
- 오픈/병합된 PR 현황
- 팀원별 PR 목록

### 5. `/dev team`
- 팀 전체 스프린트 현황 비교
- 할당량, 완료율, 커밋 활동 통합

### 6. `/dev stats <name>`
- 개발자 통계 히스토리
- 스프린트별 완료 이슈, 포인트

### 7. `/dev workload`
- 팀원별 작업 부하 분석
- 오픈 이슈 수, 포인트

## 데이터 통합

| 데이터 | 소스 |
|--------|------|
| 할당 이슈 | `jira_issues` (assignee) |
| 커밋 | `bitbucket_commits` (author_name) |
| PR | `bitbucket_pullrequests` (author_name) |
| 멤버 통계 | `member_stats` |
| 팀원 정보 | `team_members` |
| Vision 역할 | `roadmap_vision_members` |
