# Report Skill 구현

## 개요
프로젝트 리포트 생성을 위한 `/report` skill을 구현했다. 스프린트 리포트, 팀 성과, 프로젝트 현황, Epic 상세, 주간 리포트, velocity 추이 등 다양한 리포트를 마크다운 형식으로 생성한다.

## 주요 기능

### 1. `/report sprint [name]`
- 스프린트 요약 (기간, 상태, 완료율)
- 이슈 breakdown (상태별)
- 담당자별 현황
- 완료된 주요 이슈
- Blockers/Risks

### 2. `/report team`
- 팀원별 성과 (이슈, 포인트, 커밋, PR)
- 팀 velocity
- Top Contributors

### 3. `/report project [key]`
- 프로젝트 전체 현황
- Epic별 진행률
- 최근 7일 활동
- 백로그 현황

### 4. `/report epic <key>`
- Epic 상세 정보
- Story 목록 및 진행률
- 개발 활동 (커밋, PR, 기여자)
- 타임라인

### 5. `/report weekly`
- 주간 완료 이슈
- 개발 활동 (커밋, PR)
- Top Contributors
- 다음 주 Focus

### 6. `/report velocity`
- 최근 6개 스프린트 velocity
- 평균, 추세, 예측 가능성
- 다음 스프린트 추천 커밋량

### 7. `/report vision [title]`
- Vision 진행 현황
- Milestone별 현황
- Epic별 진행률

## 출력 형식
- 모든 리포트는 **마크다운 형식**
- AI가 쿼리 결과를 분석하여 리포트 문서 작성

## 데이터 통합
- Jira: 이슈, 스프린트, Epic
- Bitbucket: 커밋, PR
- Roadmap: Vision, Milestone

## 결과
